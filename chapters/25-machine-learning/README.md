# Part 25：機器學習基礎 Machine Learning Foundations

> 機器學習讓程式**從資料學會規則**,而非由你寫死規則。這一 Part 從「什麼是 ML、何時該用」講起,帶你走完一個嚴謹的 ML 工作流:資料準備、特徵工程、訓練回歸/分類模型、正確評估、對抗過擬合——建立 **ML Engineer** 的地基。重點不只在「會呼叫 `fit`」,而在**理解每一步為什麼這樣做、哪裡容易錯**。

> 🧭 屬「**資料 / AI 學習線**」,對應 **Machine Learning Engineer**。前置:[Part 17 資料處理](../17-data-science/README.md)(numpy/pandas、[ML 入門](../17-data-science/README.md))、[Part 24 統計](../24-business-analytics/README.md)。範例用 scikit-learn + numpy(固定 `random_state` 確保可重現),CI 已含 `data` 依賴。
