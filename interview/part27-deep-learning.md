## Q1. 一個神經元在做什麼?為什麼需要非線性激活函式?

**考點**:神經網路基礎([01-neural-network-basics](../chapters/27-deep-learning/01-neural-network-basics.md))

**答**:單一神經元 = **加權和 + 偏置 + 激活函式**(`a = activation(w·x + b)`)——**等同帶激活的邏輯回歸**。

**為何需要非線性激活**:若沒有(或只用線性激活),**多層網路會塌縮成單層線性模型**(線性的組合還是線性)——表達力不會超過單層邏輯回歸。非線性激活才讓多層能學**彎曲、複雜**的函式。

**追問**:常見激活——**ReLU(現代預設,`max(0,z)`)**、sigmoid/tanh、softmax(多分類);**前向傳播** = 輸入層層做「矩陣乘 + 激活」到輸出得預測;參數(W、b)就是網路的「知識」。

## Q2.(必考)梯度下降和反向傳播是什麼?

**考點**:反向傳播([02-backpropagation](../chapters/27-deep-learning/02-backpropagation.md))

**答**:

- **梯度下降**:往**負梯度方向**迭代更新參數以最小化損失(`w ← w − lr × ∂L/∂w`)——梯度定方向與步幅(陡處走大步、平處走小步)。
- **反向傳播**:用**連鎖律從輸出往輸入反向**、**一次掃描高效算出所有參數的梯度**(不必對每個參數各算一次)。

**學習率**是最關鍵超參數:**太大發散**(衝過頭越跑越遠)、太小太慢。

**追問**:**梯度檢查**——用**數值梯度**(`(L(w+ε)−L(w−ε))/2ε`)對照解析梯度,驗證反向傳播寫對;mini-batch SGD(用小批資料估梯度,快且噪音助跳出局部最優);非凸只保證局部最優。

## Q3.(必考)為什麼 XOR 需要隱藏層?訓練的完整循環?

**考點**:手刻 NN([03-nn-from-scratch](../chapters/27-deep-learning/03-nn-from-scratch.md))

**答**:XOR(相同→0、相異→1)是**非線性可分**的——**單層(邏輯回歸)只能畫直線,永遠學不會**。加隱藏層後,網路能先把輸入**非線性轉換**成一個「線性可分的表示」,輸出層再用一條線分開。**隱藏層的作用就是學一個讓問題變簡單的表示**。

**訓練循環**(所有深度學習的骨架):**前向傳播 → 算損失 → 反向傳播 → 更新參數**,重複 epoch。

**追問**:**反向傳播的誤差流動**——輸出層誤差**透過權重轉置傳回隱藏層**,連鎖律逐層算梯度;**權重要隨機初始化**(全 0 會讓神經元對稱、學一樣的東西)。

## Q4. 深度學習框架的核心價值?autograd 是什麼?

**考點**:框架([04-frameworks](../chapters/27-deep-learning/04-frameworks.md))

**答**:核心價值:**autograd(自動算梯度)、GPU 加速、現成層/優化器/生態**。

**autograd 原理**:**前向傳播時記錄計算圖**(每個運算與輸入),呼叫 `loss.backward()` 時**反向走這張圖、套連鎖律算出所有梯度**——就是**自動化的反向傳播**(不是魔法,就是你手刻反向傳播的自動版)。

訓練骨架對照手刻:`model(X)`(前向)/ `loss.backward()`(反向)/ `optimizer.step()`(更新)= 手刻四步。

**追問**:**`zero_grad()` 必要**——PyTorch 梯度是**累加**的,每輪要先清零(否則疊加,常見坑);懂原理才能 debug 框架。

## Q5.(必考)CNN 為什麼適合影像?什麼是參數共享?

**考點**:CNN([05-cnn](../chapters/27-deep-learning/05-cnn.md))

**答**:CNN 適合影像因為**利用空間結構、參數共享、局部感受野**——全連接處理影像會**參數爆炸**(200×200 圖 × 1000 神經元 = 上億權重)且丟失空間關係。

**卷積運算**:一個小 **kernel(濾鏡)滑過整張影像**,逐元素乘再求和——偵測局部特徵(某位置越像 kernel 的模式,反應越強,本質是相似度)。堆疊多層學**層次特徵**(邊緣 → 形狀 → 物體)。

**參數共享**:**同一個 kernel 用於所有位置**——參數從千萬降到極少(一個 3×3 kernel 才 9 個),且**平移不變**(特徵在哪都能偵測)。

**追問**:**池化(max pooling)** 降維 + 增位移不變性;kernel 的值是**訓練學出來的**;遷移學習(用預訓練骨幹)。

## Q6.(必考)注意力機制是什麼?怎麼比 RNN 好?

**考點**:注意力([06-sequence-attention](../chapters/27-deep-learning/06-sequence-attention.md))

**答**:序列模型演進:**RNN**(逐步處理、遠距依賴衰減、**無法平行**)→ **attention**(直接關聯所有位置、**可完全平行**)。

**scaled dot-product attention**:

```text
Attention(Q, K, V) = softmax(Q·Kᵀ / √d_k) · V
```

- **Q(query)**「我想找什麼」、**K(key)**「有什麼可被找」、**V(value)**「實際內容」。
- 用 Q·Kᵀ 算**相似度** → softmax 轉**權重** → 加權求和 V。

**為何強**:每個元素**直接關注所有相關元素**——**遠距依賴一步到位**(不隨距離衰減)、**完全平行**(一次矩陣乘算完所有位置關係,適合 GPU,是 LLM 能規模化的關鍵)。

**追問**:**縮放 `/√d_k`**——防大維度時 softmax 飽和、梯度健康;self-attention(Q/K/V 同源)+ 位置編碼(注意力本身無順序);**transformer 是所有 LLM 的地基**。

## Q7. 對抗過擬合和梯度問題有哪些訓練技巧?dropout 怎麼運作?

**考點**:訓練技巧([07-training-techniques](../chapters/27-deep-learning/07-training-techniques.md))

**答**:

**對抗過擬合**:dropout、L2 正則化、early stopping、資料增強。

**dropout**:訓練時**隨機關閉一部分神經元**——逼網路不依賴特定神經元、學穩健表示(像集成)。**inverted dropout** 把保留的放大 `1/(1−rate)` 補償;**推論時全開**(用全部知識)。

**對抗梯度消失/爆炸**:**好初始化(He/Xavier,讓各層訊號尺度穩定)、ReLU、batch norm、殘差連接**(讓極深網路訓得動)。

**追問**:**Adam vs SGD**——Adam 自適應學習率 + 動量,收斂快、對 lr 較不敏感(現代預設);學習率排程(warmup + decay);early stopping(監控驗證損失,停在泛化最佳點)。

## Q8. 通用神經網路類別怎麼設計?深度學習仍遵循 ML 原則嗎?

**考點**:通用網路([08-capstone-nn](../chapters/27-deep-learning/08-capstone-nn.md))

**答**:組件:**依 `sizes`(如 `[2,16,8,1]`)建各層參數、迴圈式 forward(快取中間值)/backward(通用處理任意層數)、mini-batch train**。

**通用反向傳播**:用**反向迴圈**逐層算梯度、把誤差往前傳——是兩層手刻版的通用化(不寫死層數,對應框架 autograd 對任意計算圖做的事)。

**深度學習仍是 ML**:train/test split、泛化、過擬合、評估原則**一以貫之**——**對比 train/test 準確率**(接近 = 泛化好、差距大 = 過擬合,加 dropout/正則化)。

**追問**:整合的訓練技巧(He 初始化、ReLU、mini-batch、標準化,缺一深層可能訓不好);懂底層才能 debug 框架、理解 LLM。
