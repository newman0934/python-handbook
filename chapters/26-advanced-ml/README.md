# Part 26：進階機器學習 Advanced Machine Learning

> [Part 25](../25-machine-learning/README.md) 打好 ML 基礎(線性/邏輯回歸、評估、過擬合),這一 Part 進入**更強的模型與模型工程**:決策樹與集成學習(隨機森林、梯度提升——實務競賽最常勝出的模型)、非監督學習(聚類、降維)、系統化超參數調校、類別不平衡處理,以及讓黑箱模型可解釋的技術。這些是 **ML Engineer** 從「會跑基礎模型」到「能打造高效能、可上線模型」的關鍵能力。
